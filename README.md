# COVID-19 Data Analysis Project

A Python-based COVID-19 data analysis project that performs state-wise reporting, recovery and death rate analysis, monthly growth visualization, and graphical representation using Pandas, NumPy, Matplotlib, and OpenPyXL.

---

# Overview

This project performs state-wise COVID-19 data analysis using Python.  
It analyzes confirmed, recovered, and deceased cases and generates graphical visualizations and Excel reports for better understanding of pandemic trends.

---

# Features

- State-wise COVID-19 report generation
- Confirmed, recovered, and deceased case analysis
- Monthly growth trend visualization
- Recovery rate and death rate analysis
- Pie chart visualization of top affected states
- Cumulative case tracking
- Excel report generation

---
